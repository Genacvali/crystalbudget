# CrystalBudget — Краткое описание для резюме

## CrystalBudget | Pet-проект

**Веб-приложение для управления личным и семейным бюджетом с AI-интеграцией**

**Технологии:** React, TypeScript, Vite, Supabase (PostgreSQL), OpenAI API, Telegram Bot API, Tailwind CSS, PWA

**Основной функционал:**
- Управление доходами и расходами с поддержкой мультивалютности
- Гибкое распределение бюджета по категориям (проценты/фиксированные суммы)
- Семейный бюджет с ролевым доступом
- AI-ассистент (GPT-4o-mini) для советов по бюджету и прогнозирования
- Telegram-бот для быстрого добавления транзакций (текст, голос, фото чеков)
- PWA с офлайн-режимом и push-уведомлениями
- Аналитика и визуализация финансовых данных

**Технические достижения:**
- Реализована сложная бизнес-логика: перенос остатков между месяцами, учет долгов по категориям, мультивалютные расчеты
- Оптимизация производительности: мемоизация вычислений, оптимистичные обновления UI, lazy loading, кэширование
- Интеграция OpenAI с Function Calling для выполнения действий в приложении
- OCR и распознавание чеков из фотографий
- 33 миграции БД, 10+ Edge Functions, полная типизация TypeScript

**Результат:** Полнофункциональное приложение, демонстрирующее навыки full-stack разработки, работу с AI-интеграциями и создание сложных пользовательских интерфейсов.
