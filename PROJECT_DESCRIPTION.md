# CrystalBudget — Описание проекта для резюме

## Обзор проекта

**CrystalBudget** — полнофункциональное веб-приложение для управления личным и семейным бюджетом с интеграцией искусственного интеллекта и Telegram-бота. Проект демонстрирует навыки full-stack разработки, работу с современными технологиями и создание сложных пользовательских интерфейсов.

## Ключевые функции

### Основной функционал
- **Управление доходами и расходами**: Отслеживание транзакций с поддержкой множественных источников дохода и категорий расходов
- **Гибкое распределение бюджета**: Настройка лимитов по категориям на основе процентов от дохода или фиксированных сумм
- **Мультивалютность**: Поддержка работы с несколькими валютами (RUB, USD, EUR и др.) с автоматическим пересчетом балансов
- **Семейный бюджет**: Совместное управление финансами с ролевым доступом (владелец/участник семьи)
- **Аналитика и отчеты**: Визуализация финансовых данных с помощью графиков и детальной статистики

### Интеграции и расширенные возможности
- **AI-ассистент (G.A.I.A.)**: Интеллектуальный помощник на базе OpenAI GPT-4o-mini для получения советов по бюджету, прогнозирования и оптимизации расходов
- **Telegram-бот**: Добавление транзакций через текст, голосовые сообщения и фотографии чеков с автоматическим распознаванием
- **PWA (Progressive Web App)**: Установка как нативное приложение с поддержкой офлайн-режима и push-уведомлений
- **Система уведомлений**: Центр уведомлений с отслеживанием важных событий бюджета

## Технологический стек

### Frontend
- **React 18** с **TypeScript** — типобезопасная разработка компонентов
- **Vite** — быстрая сборка и hot module replacement
- **React Router** — клиентская маршрутизация
- **TanStack Query (React Query)** — управление серверным состоянием и кэширование
- **Shadcn/UI** + **Tailwind CSS** — современный UI с адаптивным дизайном
- **Recharts** — визуализация финансовых данных
- **React Hook Form** + **Zod** — валидация форм
- **date-fns** — работа с датами

### Backend
- **Supabase** (PostgreSQL) — база данных и аутентификация
- **Supabase Edge Functions** (Deno) — серверная логика и API
- **Row Level Security (RLS)** — безопасность на уровне базы данных

### Интеграции
- **OpenAI API** (GPT-4o-mini) — AI-чат, анализ изображений, транскрипция голоса
- **Telegram Bot API** — интеграция с мессенджером для быстрого ввода транзакций
- **CloudPayments** — обработка платежей для подписок

### Инструменты разработки
- **TypeScript** — строгая типизация
- **ESLint** — линтинг кода
- **PostCSS** + **Autoprefixer** — обработка CSS
- **Service Worker** — кэширование для PWA

## Технические достижения

### Архитектура и производительность
- **Оптимистичные обновления UI** — мгновенный отклик интерфейса при добавлении транзакций
- **Мемоизация вычислений** — оптимизация сложных расчетов бюджета с помощью `useMemo` и `useCallback`
- **Lazy loading** — динамическая загрузка страниц для улучшения времени первой загрузки
- **Кэширование данных** — стратегия кэширования в Edge Functions для снижения нагрузки на БД
- **Rate limiting** — защита API от злоупотреблений

### Сложная бизнес-логика
- **Мультивалютные расчеты**: Автоматический пересчет балансов при работе с разными валютами
- **Перенос остатков**: Логика переноса неиспользованных средств между месяцами
- **Учет долгов по категориям**: Отслеживание превышений бюджета с переносом на следующий период
- **Пропорциональное распределение**: Расчет расходов по категориям при множественных источниках дохода

### Интеграции
- **Telegram Web App**: Авторизация через Telegram с использованием Web App API
- **AI с функциями (Function Calling)**: Интеграция OpenAI с возможностью выполнения действий в приложении (создание категорий, анализ бюджета)
- **OCR и распознавание чеков**: Автоматическое извлечение данных из фотографий чеков
- **Голосовой ввод**: Транскрипция голосовых сообщений в текст для быстрого добавления транзакций

### UX/UI
- **Адаптивный дизайн**: Полная поддержка мобильных устройств и планшетов
- **Темная/светлая тема**: Переключение тем с сохранением предпочтений
- **Скелетоны загрузки**: Улучшение воспринимаемой производительности
- **Пустые состояния**: Интуитивные подсказки для новых пользователей
- **Онбординг**: Интерактивные гайды для быстрого старта

## Масштаб проекта
